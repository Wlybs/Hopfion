# Hopfion 频率证据状态表

日期：2026-06-12

本文件是方案 B 执行期间的当前证据入口。较早文档中把 `srcX 200 GHz` 称为“候选 resonant coupling frequency”的表述，属于 ringdown 完成前的阶段判断，不能覆盖后续证据。

## 已有事实

| 频率或窗口 | 已有证据 | 当前允许表述 |
|---|---|---|
| `173.66 GHz` | 弱 `Bz` sinc 脉冲后，table-average `m_z` 与 `E_total` 自由振荡谱均出现主峰 | table-average ringdown 候选本征频率；空间归属待方案 B 检验 |
| `srcX 200 GHz` | 连续驱动下位移和后瞬态正 `dE_total/dt` 均较强；与 `173.66 GHz` 相差 `26.34 GHz` | 连续驱动强响应/能量趋势窗口；尚未确认本征共振 |
| `srcZ 100 GHz` | 异常 `+z` 位移；后瞬态 `dE_total/dt<0`；最近弱 ringdown 峰 `77.64 GHz`，相差 `22.36 GHz` | 特殊非平衡响应窗口，不是正吸收峰或已确认本征频率 |
| `srcX 1000 GHz` | 强位移，长时能量斜率符号与窗口依赖明显 | 高频强驱动、传播或散射窗口 |
| `srcZ 1100 GHz` | 强反向位移，强驱动阶段可伴随 Hopfion 失稳 | 高频强控制/非线性窗口，不是已确认本征频率 |
| 点源 `700/800 GHz` | 响应峰低于平面源 `1000/1100 GHz` | 点源有效响应红移现象；宽 k 谱解释仍待直接检验 |

## 运行中发现的边界淬火风险

2026-06-12 的临时只读审计显示，在共同的 `5-112 ps` 时间窗内，`1 mT` 与 `5 mT` 的 `m_z` 锁相幅值分别约为 `1.1633e-6` 和 `1.1628e-6`。既有 `Bx 5 mT` 与 `Bz 5 mT` 的 `m_z` 主峰功率也分别约为 `1.803e-10` 和 `1.884e-10`。这些值不呈现预期的弱场幅度标度，并且对驱动方向不敏感。

因此，`173.66 GHz` 当前进一步降级为“边界淬火可能激发的候选自由模”。它仍可能是 Hopfion 或体系的真实本征模，但现有数据不足以证明由 sinc 脉冲线性激发。正式结论等待零场控制和目标边界平衡态差分谱。

## 方案 B 的判定顺序

1. `1/2/5 mT` 的 `173.66 GHz` 峰功率若近似满足 `P proportional B0^2`，支持小振幅线性响应；若明显偏离，记录非线性或数值窗口依赖。
2. `1 ns` 数据给出峰位、FWHM 和 Q。若半高宽无法分辨，保留 `fwhm_unresolved`，不人工拟合出 Q。
3. 空间复模态必须同时满足拓扑掩膜内外功率比 `>=2` 与 Hopfion/均匀背景总功率比 `>=3`，才通过本项目局域性阶段门。
4. 圆偏结果只用于判断手性选择性。它不能单独证明模式是 translation、twist 或 breathing。
5. 只有阶段门通过后，才检查传播自旋波在 `150-200 GHz` 的锁相响应是否于 `173.66 GHz` 附近增强。
6. `E_total` 后瞬态斜率始终称为能量趋势，不单独称为净吸收率。

## 结果分支

| 观测组合 | 允许结论 |
|---|---|
| 线性标度通过，空间门通过，CW 峰与 ringdown 对齐 | 局域 Hopfion 候选本征模与传播自旋波发生候选共振耦合 |
| 线性标度通过，空间门通过，但 CW 不对齐 | 存在局域候选本征模，但该传播源几何下耦合弱或近似暗模 |
| table 峰存在，但空间门失败且均匀背景同样强 | 更可能是背景、盒子或体自旋波模式，不能归于 Hopfion |
| 点源谱熵更大，但峰值 k 不降低 | 点源造成角分布或带宽展宽，不支持“k 峰红移” |
| 点源谱熵更大且峰值 k 明确降低 | 支持宽 k/低 k 权重参与有效响应红移，但仍需与有 Hopfion 散射谱交叉验证 |
