# Hopfion 本征模与自旋波耦合机制设计

## 目标

围绕现有 weak-sinc ringdown 的 `173.66 GHz` 候选峰，依次回答四个问题：

1. 峰位是否处于弱场线性响应区，并能给出可重复的线宽和品质因数。
2. 该峰是否空间局域于 Hopfion，而不是均匀背景、有限盒子或吸收边界模式。
3. 该模式是否具有可由圆偏振区分的手性 twisting/gyrotropic 成分。
4. 连续传播自旋波能否在相同频率附近产生增强的相干响应，以及点源响应差异是否对应更宽的波矢谱。

任何阴性结果都保留。只有通过对应证据门，才使用“本征模”“共振耦合”或“点源低 k 红移机制”等术语。

## 总体结构

研究分为五个相互衔接的批次。批次 1 至 3 构成本征模证据链，批次 4 和 5 构成传播自旋波耦合证据链。所有仿真严格复用同一 Q_H=1 初始态和 frustrated-FM 参数，分析功能优先扩展 `scripts/resonance_analysis.py`，子项目脚本只负责组织输入输出。

### 批次 1：线性区与频谱精度

- 复用现有 `Bz, 5 mT, 0.5 ns` ringdown。
- 新增 `Bz, 1 mT, 0.5 ns`、`Bz, 2 mT, 0.5 ns`。
- 新增 `Bz, 1 mT, 1.0 ns`，仅用于提高峰位和线宽分辨率。
- 相同 0.5 ns 窗口比较峰位、峰功率和 `P/B0^2`。
- 1.0 ns 谱使用半高宽估计 `Q=f0/FWHM`；若峰形不适合洛伦兹型，只报告数值 FWHM，不强行拟合。

通过标准：`1/2/5 mT` 主峰在频率分辨率允许范围内稳定，且峰功率对 `B0^2` 的偏离可量化。未通过时标记为非线性候选峰。

### 批次 2：Hopfion 局域空间模态

- 两组 `Bz, 5 mT, 0.3 ns` 宽带脉冲：Hopfion 初始态与均匀 `+z` 背景。
- table 采样 `0.05 ps`；中心 `48^3` ROI 的完整磁化矢量采样 `0.2 ps`。
- 逐胞流式傅里叶投影，不把全部帧堆入内存。
- 同一数据集提取 `173.66/126.67/77.64/38.82 GHz` 附近的复振幅。
- 输出 `m_z` 振幅、`m_x/m_y` 复相位、径向局域化曲线、Hopfion 与背景功率比。

通过标准采用操作性阈值，而不是事后目测：初态拓扑掩膜内的平均谱功率至少为掩膜外的 `2` 倍，且 Hopfion run 在该频率的 ROI 总功率至少为均匀背景 run 的 `3` 倍。两个阈值均需通过。阈值是本项目的保守判据，不冒充普适常数。未通过时把该峰降级为背景或有限几何候选模式，并停止以它为中心的连续波精扫。

### 批次 3：圆偏振选择性

- 两组 table-only 高斯包络圆偏微波：`B+(t)=env(t)[cos(wt),+sin(wt),0]` 与 `B-(t)=env(t)[cos(wt),-sin(wt),0]`。
- 载频 `174 GHz`，`B0=2 mT`，包络中心 `60 ps`、标准差 `20 ps`，`T=0.5 ns`、`dt_table=0.05 ps`。
- 比较 `m_+=mx+i my` 与 `m_-=mx-i my` 的频谱。

判据：若某一旋向对特定峰有稳定增强，可描述为圆偏选择性或手性动态通道；若无差异，客观记录未检出选择性。

### 批次 4：连续传播自旋波桥接

仅在批次 2 通过后运行。

- 主扫描：`srcX-vibX`，`B=0.2 T`，`f=150/160/170/174/180/190/200 GHz`，每组 `0.3 ns`。
- 振幅对照：`f=174 GHz`，`B=0.05/0.1 T`；`0.2 T` 由主扫描复用。
- 几何对照：`srcZ-vibX` 与 `srcX-vibZ`，均为 `174 GHz, 0.2 T`。
- table 采样 `0.1 ps`，中心 ROI 或稀疏全场 OVF 只用于位移/形变验证，避免高频全场输出。

判据同时使用驱动频率锁相响应、平均位移、核心形变和与 ringdown 峰的频差。`E_total` 后瞬态斜率仅记录为能量趋势，不称为净吸收率。只有弱驱动相干响应峰与 ringdown 峰对齐时，才写作“本征共振耦合”。

### 批次 5：点源与平面源波矢谱

- 无 Hopfion 均匀背景，避免散射体本身污染入射波谱。
- 四组短时仿真：平面源/点源分别在 `700 GHz` 与 `1000 GHz`。
- 保持相同源位置和极化方向。平面源为 `0.1 T`、点源为 `10 T`；平面源包含 `10000` 个源格点，因此两者满足相同的 `sum(B^2)` 源强范数。比较时仍对谱做总功率归一化，不比较绝对幅度。
- 总时长 `30 ps`，切片保存间隔 `0.02 ps`，保存穿过源与 Hopfion 原位置的二维切片。
- 对空间维度做 FFT，提取谱宽、峰值 k 和低 k 功率占比。

判据：点源相对平面源必须显示可重复的更宽 k 分布及更高低 k 权重，并且峰值 k 确有降低，才支持“有效红移”。若峰值 k 相同而谱熵增大，只解释为角分布或带宽展宽。

## 数据与错误处理

- 每个 mx3 在运行前执行 `mumax3 -vet`。
- Python 前置检查固定为 Hopfion venv 与 `import discretisedfield`。
- 仿真日志只用文件重定向，不使用 `tee`。
- 每个批次生成机器可读 manifest，记录脚本哈希、参数、开始/结束时间、表格末时刻和退出码。
- 分析遇到缺帧、非单调时间或峰宽触及搜索边界时返回未通过状态，不填补或外推数据。
- 大数据只保留必要 ROI/切片；原始数据与派生图保持在结果包内。

## 预期论文逻辑

最终论文叙事按证据强度分层：自由振荡峰给出候选本征频率，空间局域图确认模式归属，圆偏振给出对称性信息，弱连续波相干响应谱检验传播自旋波耦合。强位移但不对齐的 `200/1000/1100 GHz` 仍称为传播、散射或强驱动窗口，不反向修改数据以迁就预期。
